Dislocation lines

This data object holds a set of dislocation lines that have been extracted from an atomistic input crystal using the Dislocation analysis (DXA) modifier, or which have been imported from a discrete dislocation dynamics (DDD) simulation.

Each dislocation is represented by a continous three-dimensional curve and is accociated with a Burgers vector and the type of crystal lattice it is embedded in. The table of dislocation lines is accessible in the data inspector panel of OVITO. Furthermore, it is possible to export the dislocation lines from OVITO to a VTK output file, for example.

Note that the visual appearance of dislocation lines in rendered images is controlled through the associated Dislocations visual element.
