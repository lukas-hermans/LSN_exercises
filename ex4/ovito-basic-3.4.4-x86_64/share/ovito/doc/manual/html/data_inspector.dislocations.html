Dislocations (data inspector)

This page of the data inspector shows a table of all dislocation lines in the current dataset. Dislocation lines may extracted from an atomistic crystal by applying the Dislocation analysis (DXA) modifier.

The crosshair button above the table activates a mouse input mode which lets you pick individual dislocation lines in the viewports. As you select lines in the viewports, the corresponding rows in the table get highlighted. Hold down the Ctrl key (Command key on macOS) to select multiple dislocation lines. This also works the other way around: With the highlighting mode active, you can select table rows and OVITO will highlight the corresponding dislocations in the viewports. Click the crosshair button again or right click in a viewport to deactivate the highlight mode.
