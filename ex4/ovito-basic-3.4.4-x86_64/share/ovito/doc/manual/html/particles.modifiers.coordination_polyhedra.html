Coordination polyhedra

This modifier computes coordination polyhedra around selected atoms for visualization purposes. A coordination polyhedron is the convex hull spanned by the bonded neighbors of a central atom.

Note: The modifier creates coordination polyhedra only for currently selected particles. That means you have to select the particles first for which you would like to generate coordination polyhedra, e.g. using the Select type modifier.

The modifier takes into account the bonds from the selected particles to their neighbor to build the polyhedra. Thus, it may be necessary to create these bonds first using the Create bonds modifier before applying the Coordination polyhedra modifier.

The coordination polyhedra generated by the modifier are so-called surface meshes and their visual appearance, for example the surface transparency, can be configured through the associated visual element appearing in the pipeline editor after applying this modifier.

Note: This modifier has no configurable parameters.
