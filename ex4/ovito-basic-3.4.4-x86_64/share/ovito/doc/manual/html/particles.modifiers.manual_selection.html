Manual selection

This modifier lets the you manually select individual particles or bonds in the viewports using the mouse. You can use it to restrict the action of subsequent modifiers in the pipeline to a selected subset of elements.

Two selection tools are available: The Pick mode lets you toggle the selection status of individual particles or bonds by picking them with the mouse. The Fence selection mode lets you draw a closed fence around a group of particles or bonds in the viewports. All elements within the polygonal path will be selected. Hold down the Ctrl key (Command on Mac) to add elements to the existing selection set. The Alt key can be used to remove elements from an existing selection set.
